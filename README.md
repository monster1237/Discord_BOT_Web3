# Discord_BOT_Web3
Discord上的查询数字货币价格机器人

环境变量
COINMARKETCAP_API

DISCORD_BOT

命令指南
输入"查询+币种名称"可获取最新价格、项目方社交网站（推特.Discord社群、电报）
